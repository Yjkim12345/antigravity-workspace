# Antigravity 지식 자동화 파이프라인 (Zero-Click Knowledge Automation)

이 프로젝트는 법률 문서, 판례, 아이디어 등을 복사하거나 메모창에 입력하는 즉시 Gemini AI가 분석하여 '한줄법리' 형태의 독립적 지식(Zettelkasten) 카드로 변환한 후 Notion 데이터베이스에 자동 저장하는 자동화 도구 모음입니다.

## 📁 주요 파일 및 구조 (프로그램 개별 분리 구조)

시스템은 유연한 유지보수와 안정성을 위해 **기능별로 3개의 주요 파이썬 스크립트**로 나누어져 있습니다. 한 파일에 오류가 발생해도 다른 기능은 독립적으로 동작합니다.

### 1. `knowledge_atomizer.py` (핵심 처리 엔진)
* **역할:** 이 파이프라인의 **중추 신경(심장)**입니다. 다른 입력 스크립트들(`hotkey`, `quick_input`)이 텍스트를 던져주면 이를 받아 Gemini API(현재 `gemini-3.1-pro-preview`)에 전송하여 분석하고, 결과물을 파싱해 Notion API로 적재합니다.
* **실행:** 단독으로 문서를 통째로 넣고 실행할 수 있습니다. `python knowledge_atomizer.py <문서파일명.txt>`

### 2. `knowledge_hotkey.py` (단축키 감지 모듈)
* **역할:** 백그라운드에서 상시 실행되며 윈도우 단축키 `<Ctrl+Alt+N>` 입력을 감지합니다.
* **작동 방식:** 엘박스나 웹 등에서 복사해 둔 클립보드 텍스트를 읽어들인 뒤, 즉시 내부적으로 `knowledge_atomizer.py` 엔진을 호출해 노션으로 전송합니다.
* **실행:** 터미널에서 `python knowledge_hotkey.py` 로 켜두면 항상 대기 상태로 작동합니다. (로그는 `hotkey.log` 참조)

### 3. `knowledge_quick_input.py` (전용 GUI 팝업 메모장)
* **역할:** 바탕화면 바로가기 등을 통해 번개처럼 띄울 수 있는 깔끔한 입력창(Tkinter)입니다.
* **작동 방식:** 사용자가 떠오른 법리나 메모를 작성하고 **`<Ctrl+S>`** 를 누르면, 창은 1초 만에 즉각 닫히지만 백그라운드에서는 `atomizer` 코어 엔진이 실행되어 노션 업로드를 끝마친 후 안전하게 종료됩니다.

## 🛠️ PC 이동 시 유실 방지 가이드 (Git 동기화)

어제 만든 코드를 오늘 다른 PC에서 다시 개발하는 비효율을 방지하기 위해, 이 폴더는 **버전 관리 시스템(Git)으로 동기화**되고 있습니다. 

어떤 PC에서든 스크립트를 수정(개선)하거나 새로운 메모장 런처를 추가했다면, 반드시 아래 규칙을 따라주세요.

### 1. 퇴근 전 / 작업 종료 시 (현재 PC)
작업이 끝났다면 동기화 폴더(`sync_workspace`)에 있는 **`sync_up.bat`** 파일을 더블클릭해서 실행하세요. 오늘 만든 코드와 문서가 모두 GitHub 리포지토리(안전 금고)에 클라우드로 영구 백업됩니다.

### 2. 출근 후 / 새 PC에서 코볼 때 (다른 PC)
새로운 PC에 앉았다면 가장 먼저 **`sync_down.bat`** 파일을 더블클릭해서 실행하세요. 어제 다른 PC에서 만들어 두었던 최신 코드(`make_shortcut.py`, `README` 등)가 즉시 복원되어 동일한 환경에서 작업을 이어나갈 수 있습니다.

## 🚀 빠른 시작 (바탕화면 단축키 팁)

* **팝업창(Quick Input) 작업표시줄 고정하기:** 
   Windows에서는 `.bat` 파일을 직접 작업표시줄에 끌어다 놓을 수 없습니다. 이 폴더에 있는 `make_shortcut.py` 를 실행하면, 작업표시줄에 즉시 고정할 수 있는 전용 바탕화면 바로가기(`새법리_입력기_작업표시줄고정용.lnk`)가 생성됩니다.
