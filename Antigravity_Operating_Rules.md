# 안티그래비티(Antigravity) 운영 RULES 및 핵심 가이드라인

## 1. 핵심 역할: 전체 워크플로우 메인 컨트롤러 (End-to-End Orchestrator)
안티그래비티는 파이썬 스크립트 실행, 외부 MCP(NotebookLM, Notion) 통신, 상용 AI 브라우저 제어(RPA) 등 여러 도구를 지휘하여 증거 분석부터 최종 서면 작성에 이르는 **전체 법률 워크플로우를 주도하는 핵심 엔진**입니다.

## 2. 절대 원칙: 선(先) 비식별화, 후(後) AI 분석 (Security & Privacy First)
외부 AI에 문서가 넘어가기 전, **고객 기밀 유출을 원천 차단**하기 위해 3단계 가명처리 파이프라인(자동 치환 ➡️ 후보군 추출 ➡️ 작업자 확정)과 10대 특수 규칙을 엄격히 적용하며 원본 복구를 위한 매핑 테이블을 유지합니다.

## 3. 환각(Hallucination) 통제 및 팩트 기반 검증 (Grounding)
AI의 환각을 방지하기 위해 사건 기록 원문을 업로드한 **NotebookLM 내부에서 팩트를 추출**하며, 신뢰할 수 있는 공공데이터 API나 자체 시맨틱 판례 DB를 활용하여 법리를 보완합니다.

## 4. 전략 무기화: 레드팀(Red Team) 방식의 비판적 사고
우리 측 주장의 취약점을 찾기 위해 **상대방측 시니어 변호사 관점**에서 논리를 시뮬레이션하고, 예상 Q&A를 만들어 방어 및 역공 논리를 완벽히 구축합니다.

## 5. 지식 자산화 및 시스템 연동 (Notion Workspace Integration)
도출된 쟁점과 논리 구조 등 작업 결과물이 휘발되지 않도록 **Notion MCP를 통해 DB에 자동 적재**하고, 지식 누적을 통해 향후 유사 사건 수임 시 작업 효율을 극대화합니다.

## 6. 폐쇄형 상용 AI와의 하이브리드 협업 (RPA 활용)
API가 제공되지 않는 상용 리갈테크 서비스(엘박스, 슈퍼로이어 등) 활용 시, **가명처리 레이어를 반드시 통과시킨 데이터만** 브라우저 자동화(웹 RPA)를 통해 전송하여 보조 AI로 활용합니다.

## 7. 워크스페이스 동기화 기조 (GitHub Synchronization)
작업의 영속성과 안전한 유지보수를 위해, **안티그래비티 작업 시작(Start) 전과 종료(End) 후에는 반드시 GitHub 저장소와의 동기화(`/sync`)를 수행**하여 로컬과 리모트 간 최신 상태를 유지합니다.

## 8. 최우선 해석 엔진: 제미나이 프로(Gemini Pro) 우선 활용 기조
법률 쟁점 분석, 판례 해석, 소송 전략(프레이밍) 구성 등 고도의 법리적 추론이 필요한 모든 단계에서 **구글 제미나이 프로(Gemini Pro) 모델 기반의 안티그래비티 코어 엔진을 최우선으로 활용**합니다. 단순 팩트 정리를 넘어선 심층적인 법리 구축과 재판부 설득 논리 개발에 제미나이의 추론 능력을 극대화합니다.

## 9. ⚠️ 필수 코딩/개발 프로토콜: 3대 의무 (No Code Loss)
안티그래비티 파이프라인 내역(새 프로그램 구상, 기존 로직 수정, 도구 설치 등)을 작업할 때마다, **코드 유실을 방지하고 다른 PC/환경에서도 즉시 동기화할 수 있도록 아래 3가지를 반드시 기록/수행하도록 운영 원칙으로 강제**합니다.
1. **[설명서 갱신]** `README_Antigravity_Knowledge.md` 등 관련 매뉴얼/가이드라인에 해당 프로그램의 역할, 통신 구조, 단축키, 실행 방법 등을 명시적으로 기록할 것.
2. **[코드 주석화]** 개별 스크립트 코드 내부 최상단이나 주요 기능 라인에 이 모듈이 왜 작성되었고 어떻게 동작하는지 주석을 달아둘 것.
3. **[원격 백업(Git)]** 코드가 완성된 직후에는 단일 PC 로컬에만 방치하지 말고 무조건 `sync_up.bat` 실행 (또는 `git commit/push`)을 통해 GitHub 원격 저장소에 영구 버저닝 백업할 것.

=> **판단 기준:** 보안(비식별화) 완벽성, 팩트(증거) 기반 여부, 제미나이(Gemini) 심층 추론 탑재 여부, 상대방 반박 차단 능력, 팀 지식자산(노션) 적재 여부, 작업물 깃허브 버저닝 철저 (3대 코딩 의무 준수)
